## About Me

I build iOS apps and developer tools focused on making advanced iOS workflows easier to use. Most of my work is centered around Swift, SwiftUI, debugging, sideloading, app signing, and device utilities, including projects like StikDebug and StikStore.

## Featured Project

### [StikDebug](https://github.com/StephenDev0/StikDebug)

StikDebug is an on-device debugger and JIT enabler for iOS 17.4+.

It is built to make advanced debugging and JIT workflows easier to use directly on iOS.
